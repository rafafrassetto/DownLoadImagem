# Projeto: Carregamento Assíncrono de Imagens em Android

## Resumo do Projeto
Este repositório contém o código-fonte de um aplicativo Android desenvolvido com o propósito de implementar e analisar o consumo de mídias externas. O sistema realiza o download e a renderização de imagens hospedadas na internet, utilizando processamento assíncrono para evitar o bloqueio da thread principal e garantir a fluidez da interface de usuário.

O projeto aplica conceitos modernos de desenvolvimento mobile, com foco na construção de interfaces declarativas e gerenciamento eficiente de memória e cache no carregamento de bitmaps.

## Tecnologias e Dependências
* **Linguagem:** Kotlin
* **Interface de Usuário:** Jetpack Compose 
* **Componentes Nativos:** LazyColumn, Scaffold
* **Biblioteca de Imagens:** Coil (Coroutine Image Loader) versão 2.4.0
* **Ambiente de Desenvolvimento:** Android Studio

## Requisitos de Execução
* SDK Mínimo: API Nível 24
* Conexão ativa com a internet (permissão `android.permission.INTERNET` declarada no Manifest).
